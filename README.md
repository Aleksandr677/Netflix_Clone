# Netflix_Clone
Repository for creating Netflix clone 

Stack:
1. MVVM
2. NO Storyboadrs
3. UIKit
4. URLSession
5. Swift Package Manager
6. CoreData
7. YouTube API
8. TMBD API
9. Google Cloud
